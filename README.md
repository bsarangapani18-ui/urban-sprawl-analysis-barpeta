🏙️ Urban Sprawl Analysis – Barpeta District
📌 Overview

This project analyzes urban expansion in Barpeta District using satellite imagery to understand land-use changes over time.

⚙️ Methodology (Simplified)
Used Landsat-7 (2000) and Landsat-8 (2022) satellite data
Data sourced from USGS Earth Explorer
Processed imagery using ArcGIS 10.7
Generated Land Use/Land Cover (LULC) maps
Applied Normalized Difference Built-up Index (NDBI)
Compared built-up area changes over time
📊 Key Findings
Significant increase in built-up area between 2000 and 2022
Rapid urban expansion observed in Barpeta region
Clear transformation from rural to urban land use
🎯 Impact

Provides insights for urban planning, land-use management, and infrastructure development.

🛠️ Tools Used
ArcGIS 10.7
Remote Sensing (Landsat Data)
Spatial Analysis Techniques
